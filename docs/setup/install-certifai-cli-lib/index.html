Install Certifai Toolkit

#Prerequisites

  • Download the Certifai toolkit
  • Python 3.6 (support for 3.7 and 3.8 coming soon)
  • Conda or another virtual environment manager (recommended)

#Install the Certifai Toolkit

  1. Create a virtual environment

    conda create -n certifai python=3.6

  2. Activate the newly created environment

    conda activate certifai

  3. Install the base requirements

    pip install -r requirements.txt

  4. Install the CERTIFAI packages

    On MAC or Linux:

    pip install packages/*

    For Windows COMMAND shell:

    for %f in (packages\*.zip) do pip install %f

    For Windows Powershell:

    Get-ChildItem .\packages\*.zip | ForEach-Object -Process { pip install $_ }

  5. Verify that the Certifai CLI and client libraries are installed successfully.

    certifai -h

    A usage statement is displayed if the Certifai CLI has been installed successfully.
