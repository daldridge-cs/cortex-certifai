View scans in the Console

The Console allows you to view visualizations for scan results generated from your scans defined locally in:

  • Certifai CLI
  • Python notebooks (using Certifai libraries)
  • Text editors (using the .yaml template)

To view scans that have been run remotely, go to the remote Certifai Console URL provided by your administrator.

#Prerequisites

  • Download the Certifai Toolkit
  • Install the Certifai Toolkit

#Open the local Certifai Console

  1. Activate the certifai conda environment
    conda activate certifai
  2. In your terminal go to the directory where you downloaded your toolkit.
    cd certifai_toolkit
  3. Start the local Certifai Console using the following command

Windows Users

The SCAN_RESULTS_DIRECTORY environment variable accepts unix style paths. So, if you want to store scan results in a Windows directory folder1\folder2\reports, you should set SCAN_RESULTS_DIRECTORY=folder1/folder2/reports

Environment Variables for Windows Users

Set the environment variables in your CMD prompt appropriately. Double quote paths if they contain any spaces

For CMD prompt, you can use the following command

set SCAN_RESULTS_DIRECTORY=<unix_style_path_to_reports> python -m certifai.console.app

MAC and Linux users can use the following command to start the Console

SCAN_RESULTS_DIRECTORY=<path-to-reports> python -m certifai.console.app

Example (displays the sample reports provided with the Certifai Toolkit):

SCAN_RESULTS_DIRECTORY=./examples/reports python -m certifai.console.app

4. The Console is available at http://localhost:8000. Copy this URL into a browser to view your scan result visualizations.
