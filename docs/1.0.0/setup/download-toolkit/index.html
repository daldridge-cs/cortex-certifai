Download Certifai Toolkit

The Certifai Toolkit is used in both remote and local implementations of Certifai, as well as for use with Jupyter lab. The toolkit includes the following:

  • Reference Model server - serves the reference models provided to help you get started with Certifai
  • Certifai CLI - tool for inputing scan definitions locally or remotely
  • Client library/API - methods that run the scans
  • Certifai Console - UI for visualizing scan results

#Prerequisites

  • Remote Certifai Console URL - requires installation of Certifai on RedHat OpenShift

#Download Certifai Toolkit

Navigate to the remote Certifai Console at the URL provided by your Administrator.

In the Certifai Console

  1. Click on the help icon at the top right.

  2. Select Download Toolkit.

  3. Accept the license agreement.

  4. The zip package is downloaded to your local drive.

  5. Make a directory to work in (e.g. certifai).

    mkdir certifai

  6. Unzip the package into it.

    • Mac/Linux users:
      unzip certifai_toolkit.zip -d certifai
    • Windows users Unzip the files to the proper location using your preferred GUI tool, or run the following prompt in POWERSHELL:
      Expand-Archive -LiteralPath certifai_toolkit.zip -DestinationPath certifai -Force
      NOTE: If a certifai_toolkit.zip exists, it will be overwritten automatically.

  7. Go into the folder to view the zip file contents.

    cd certifai

    The zip file contains:

    • a packages folder containing a set of python packages as zip files
    • a pip requirements.txt file
    • an examples folder containing scan definitions and datasets for:
      • the 'healthcare disease prediction (diabetes)' example model use case
      • the 'banking loan approval (german credit)' example model use case.

#Next steps

  • For users who want to get acquainted with the scan results visualizations: open the Console and view the sample reports available with the toolkit.

  • For users, like data engineers, who have developed models and who want to quickly define a scan from a command line: Install the Certifai CLI.

  • For users who are developing models and who are comfortable using python notebooks, like data scientists: Install the Certifai libraries.
