Get started in Jupyter Notebooks

#Prerequisites

  • Download the Certifai toolkit
  • Install the client libraries
  • Add the notebook requirements (Jupyter etc) using the requirements.txt file in the notebooks directory: pip install -r examples/notebooks/requirements.txt

#Run the Scan Example notebook

  1. Change into the notebooks directory

    cd examples/notebooks

  2. Start the Jupyter server

    jupyter notebook

    A file list is displayed in the browser.

  3. Select an example notebook (see below for details).

  4. Click on Run to execute each cell in turn. This illustrates how to scan a model using a scan definition yaml file. You can view the definition file in the examples/definitions folder.

    When the notebook run completes, the scan report is returned as JSON in the last cell.

#Example Notebooks

#Processing a predefined scan definition

Open the notebook ScanExample.ipynb. This illustrates the use of Certifai to generate reports based upon an existing scan definition, with multiple models and multiple analysis types which can be run as a single evaluation.

#Creating scans of locally defined models

Open the notebook CleanStart.ipynb. This illustrates the use of Certifai to create and run a fairness scan of a locally defined model from first principles.

#Direct Certifai engine integration with a local model

Open the notebook EngineExample.ipynb. This illustrates the use of Certifai to find counterfactual explanations of a locally defined model (which is in the notebook) in the German credit use case.

#Visualizing scan results

View the report visualizations in your local Console.

MAC/Linux:

`SCAN_RESULTS_DIRECTORY=<path-to-reports> python -m certifai.console.app`

Windows Users

The SCAN_RESULTS_DIRECTORY environment variable accepts unix style paths. So, if you want to store scan results in a Windows directory folder1\folder2\reports, you should set SCAN_RESULTS_DIRECTORY=folder1/folder2/reports

_Environment Variables for Windows Users_
Set the environment variables in your CMD prompt appropriately. Double quote paths if they contain any spaces

For CMD prompt, you can use the following command

set SCAN_RESULTS_DIRECTORY=<unix_style_path_to_reports>
python -m certifai.console.app
